# Quick Start Guide

Get your CYM Studio website up and running in 5 minutes!

## 1. Install Dependencies

```bash
npm install
```

## 2. Run the Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) 🎉

## 3. Test Everything

- ✅ Use arrow keys (or WASD) to navigate around the spaceship
- ✅ Hover over TVs to see tournament descriptions
- ✅ Click TVs to watch videos
- ✅ Click to shoot lasers at incoming asteroids
- ✅ Check out the world clocks (CET, EST, KST)
- ✅ Click the Menu button to view sidebar sections

## 4. Deploy to Production

### Deploy to Vercel (Recommended)

1. Push your code to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Import your repository
4. Deploy! 🚀
