# CYM Studio - Interactive Spaceship Experience

An immersive, interactive web experience featuring an AI-powered video production studio presented as a futuristic spaceship. Explore the studio, watch tournament videos, and defend against asteroids!

![CYM Studio](public/cym.png)

## 🚀 Features

- **Interactive Spaceship Environment** - Explore a circular spaceship with your avatar
- **4 Curved TV Screens** - Watch Starcraft tournament videos on hull-mounted displays
- **Asteroid Defense System** - Shoot down incoming asteroids with laser turrets
- **World Clocks** - Track time in CET, EST, and KST
- **Mobile-Friendly** - Full touch controls and responsive design
- **Real-time Effects** - Shield impacts, explosions, and dynamic animations

## 🎮 Controls

### Desktop
- **Arrow Keys / WASD**: Move avatar
- **Mouse**: Aim laser turret
- **Click**: Shoot lasers
- **Walk near TVs**: Trigger video playback

### Mobile
- **Drag**: Move avatar
- **Tap**: Shoot lasers
- **Tap TVs**: Watch videos

## 🛠️ Tech Stack

- **Next.js 14** - React framework with App Router
- **TypeScript** - Type-safe development
- **Canvas API** - 2D graphics and animations
- **CSS Modules** - Scoped styling

## 📦 Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```
